Mycee Accessories Stock Control System - Portable Version

To run the server, choose one of these options:

1. Double-click "Mycee_Server_Launcher.bat" (Windows Batch)
2. Double-click "Mycee_Server_Launcher.ps1" (PowerShell)
3. Run manually: python server.py
4. Production mode: python server.py (with FLASK_ENV=production)

Access the system at: http://127.0.0.1:5000
